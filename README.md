游戏介绍:
    
    由当前泡泡出发，经过若干个相邻泡泡，存在可连到顶端的通路，则认为该泡泡具有挂点，不会下坠。当泡泡失去挂点则下坠。
    
    泡泡的消除：主要有爆掉和下坠两种方式，前者是后者的基础。
  
    泡泡爆掉：当玩家主动令游戏区形成 3 个或以上的同色泡泡相连，则这些泡泡将爆掉。
  
    泡泡下坠：当爆掉的泡泡造成连接的异色泡泡失去挂点，则这些异色泡泡将下坠，算入消去的泡泡中。（下坠中的泡泡不对其他泡泡发生作用）
